Documentation Scripts Deel-C RWS
========================================================================
Welcome to Uitwerking BPKV-criteria Scripts Deel-C 2022's documentation!
This package contains Python scripts developed by Waardenburg Ecology commissioned by Rijkswaterstaat (RWS).
The scripts automatically generates figures and tables for the protocol of 'Deel-C'.
In this way, the required parameters and data analysis are standardized and automated for the measurements of Macrozoobenthos.
